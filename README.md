# 🌐 GitHub User Finder (Поиск пользователей GitHub)

## 👤 Автор
Абдуллина Самира


## 📝 Описание
Графическое приложение на Python для поиска пользователей по логину с использованием официального GitHub API. Приложение отображает основную информацию о пользователе, позволяет сохранять найденные аккаунты в список избранного и восстанавливает его при следующем запуске.

### ✨ Возможности:
- 🔍 Мгновенный поиск по логину через GitHub REST API
- 📋 Вывод результатов в структурированном списке
- ⭐ Добавление и удаление пользователей из избранного
- 💾 Автоматическое сохранение избранного в `favorites.json`
- ✅ Валидация ввода и обработка сетевых ошибок
- 🛡 Только стандартные библиотеки Python (без `pip install`)
